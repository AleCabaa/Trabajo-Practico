Sentencias Usadas en GIT

Git es un sistema de control de versiones muy poderoso y versátil que incluye muchas sentencias útiles para administrar y trabajar con repositorios de código.

GIT INIT

Inicia un nuevo repositorio Git en un directorio local.

GIT CLONE

Clona un repositorio Git existente desde una URL remota.

GIT ADD

Agrega cambios en archivos al área de preparación (staging).

GIT COMMIT -M

Crea un nuevo commit con los cambios en el área de preparación y un mensaje descriptivo.

GIT STATUS

Muestra el estado actual del repositorio, incluyendo archivos modificados, en el área de preparación y sin seguimiento.
